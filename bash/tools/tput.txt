man terminfo

setaf: set the forground to color by number
setab: set background color by number
setf: set the forground to color by number
setb: set background color by number
	The difference in these caps is the color order
	seta*:
		black red green yellow blue magenta cyan white
		?The a stands for ANSI?
	set*:
		black blue green cyan red magenta yellow white

cols: the number of columns
colors: teh number of colors
