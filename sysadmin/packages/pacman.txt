=========
-0- about
=========
pacman is the package manager for arch linux.  It generally handles the
official arch linux repositories.  However, Arch also has the AUR which
is for user-managed packages (check and verify install scripts to ensure
no viruses etc if using AUR...)

===========
-1- options
===========
	-Q query database
		checks
			-d
				restrict to search deps
			-t
				restried to unrequired deps
	-R --remove
		-s
			remove recursively (also remove deps etc)

	-S sync
		synchronzies with remote repos
		-s
			search
		-u
			update packages
		-y
			Update package database

========
-2- keys
========
Update the archlinux-keyring package
probably need to update archlinux-keyring package before
refresh-keys or anything?

Keys are located in `/etc/pacman.d/gnupg`
To reset keys:
	delete the keys directory above
	then:
		pacman-key --init
		pacman-key --populate

refresh keys:
	pacman-key --refresh-keys

=======
-4- AUR
=======

steps:
-------------------
acquire the source:
-------------------
git clone https://aur.archlinux.org/package_name.git
or download a snapshot on the aur page
------
verify
------
cd pkgname
less PKGBUILD
search for .sig/.asc etc
-----
build
-----
makepkg
	-s/--syncdeps     install deps
	-i/--install      install on success
	-r/--rmdeps       uninstall deps (deps may be required for upgrade etc)
-------
install
-------
pacman -U name.pkg.tar.zst
