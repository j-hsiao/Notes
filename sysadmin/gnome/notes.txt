------------------------------
-0- gsettings
------------------------------
gsettings is basically a key-value database for settings.
It is used by various applications like gnome apps.

------------------------------
-1- commands
------------------------------
gsettings is commandline interface for gnome settings.
It seems like the settings ui does not show all the possible settings

Generally, settings are stored in a schema[:path] format.
schema is generally a . delimited path
ex: org.gnome.desktop.wm.keybindings

gsettings uses subcommands:
	help                      Show this information
	list-schemas              List installed schemas
	list-relocatable-schemas  List relocatable schemas
	list-keys                 List keys in a schema
	list-children             List children of a schema
	list-recursively          List keys and values, recursively
		schema
	range                     Queries the range of a key
		schema key
	describe                  Queries the description of a key
		schema key
	get                       Get the value of a key
		schema key
	set                       Set the value of a key
		schema key value
	reset                     Reset the value of a key
	reset-recursively         Reset all values in a given schema
	writable                  Check if a key is writable
	monitor                   Watch for changes
