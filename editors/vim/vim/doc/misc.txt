*misc.txt*	Plugin with miscellaneous settings.

Miscellaneous key bindings and settings.  If keys are already bound,
then the mappings will not be made.

===normal bindings===
<Leader>?	normal mode leader bindings
<C-[><C-[>	:nohl
<Leader>b	create a scratch buffer
gq	wrap base gq to allow a count specifying width.

===insert bindings===
<Leader>?	insert mode leader bindings
<Leader><Leader>	original <Leader> binding/key
<Leader>r	repeat a character
<C-L>	delete to the right
;l	<Esc> without having to reach for ctrl

===abbreviations===
Inp	Inputs
	======
Outp	Outputs
	=======
