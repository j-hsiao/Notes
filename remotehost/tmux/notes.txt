	=================
	-0.1- terminology
	=================
	Session
		A collection of windows.

	Window
		a "space" that takes the entire "screen"
		The terminal can only display a single window at a time.

	Pane
		A splitting of a "Window"

	Key sequence
		A sequence of keys that generally activate a command
		binding.  Key sequences will be separated by a space.
		A dash will indicate combination with a modifier key.
		Example: "C-b [" means press control+b and then [.

	Prefix key
		All tmux keybindings are prefixed with the "prefix key".
		In these notes, the prefix key will be omitted unless otherwise
		stated referred to as <pre>

	C-x/M-x
		C-x means press Control + the x character.
		M-x means either pressing alt + x, or pressing Esc followed by x.
		Both seem to activate the M-x bindings.

============
-1- commands
============
Tmux implements its own scripting language which is very similar to bash.
Note that when calling tmux commands from the shell, the shell first parses
arguments, and then tmux.  This means that to chain tmux commands, the semicolon
must be escaped.

____________
same as bash
	Words are split by whitespace.
	The first word is a command, the rest are arguments.
	semicolon/new-line terminates a command.
	Quoting is done with '', ""
	"" allows substitutions, '' does not.
	# starts a comment
	Line continuation with trailing \

______________
diff from bash
	{} is parsed as config file and then converted to string and can be nested.
	commands separated by a ; is more like bash &&
	Line continuation is allowed everywhere except in {}

	?substitutions are only inside double quotes? double quotes are like merging fo bash "" and $'' syntax?
		Get environment variable with $env_var
		~user is replaced with their home dir, defaulting to current user
		\uXXXX \uXXXXXXXX for hex unicode
		\e \r \n \t  for esc, carriage return, newline, tab
		\ooo for octal
		\C literal C character for any beside above mentioned ones.

=======================
-1.1- Common flags/args
=======================
	====================================
	[-t|-s] [client|session|window|pane]
		These specify target client/session/window/pane for a command
		______
		client
			The pty (/dev/...) of any currently attached client.
			In the shell, `tmux list-clients` or `tty`. NOTE list-clients only shows
			currently attached client sessions.
		_______
		session
			Checked in order:
				=exactmatch
				$ID
				session-name
				start of session-name
				glob pattern
			Error if ambiguous
		______
		window
			session:window where session is as above, and window is (in order):
				=exactmatch
				special token
				window index
				window id @id
				exact window name
				window name prefix
				glob pattern
			empty indicates next unused index or current window depending on the command.
			______________
			special tokens
				long            meaning
				{start} (^)     lowest window index
				{end} ($)       highest window index
		____
		pane
			window[.id] where window is as above.  Empty id means the active pane.
			______________
			special tokens
				{top}             The top pane
				{bottom}          The bottom pane
				{left}            The leftmost pane
				{right}           The rightmost pane
				{top-left}        The top-left pane
				{top-right}       The top-right pane
				{bottom-left}     The bottom-left pane
				{bottom-right}    The bottom-right pane
				{up-of}           The pane above the active pane
				{down-of}         The pane below the active pane
				{left-of}         The pane to the left of the active pane
				{right-of}        The pane to the right of the active pane
		_____________________
		common special tokens
			{last} (!)        the previously selected item by time
			{next} (+)        the next item by index
			{previous} (-)    the previous window by index
				+- can be followed by a count
			{mouse} (=)       item with the most recent mouse activity
			{marked} (~)      marked item
		________
		item IDs
			$session
			@window
			%pane
	============
	shell-commad
		This is just a single string as would be passed to `sh -c shell-command`
		or separate arguments of a command if supported

sessions
	attach
	attach-session [-dErx] [-c working-directory] [-f flags] [-t target-session]
		creates a new client if outside tmux.  Otherwise modifies current client.
		-d: detach other clients from this session
		-x: send SIGHUP too
		-f: comma-delimited client flags
			active-pane   client's active pane does not affect other clients
			ignore-size   don't affect other clients' size
			no-output
			pause-after=seconds
			read-only
			wait-exit
		-r: alias for `-f read-only,ignore-size`
		-c: set cwd for new windows
		-E: do not use update-environment option

	detach
	detach-client [-aP] [-E shell-command] [-s target-session] [-t target-client]
		detach the current client.
		-t target-client
		-s target-session
		-a all clients except -t
		-P send SIGHUP
		-E run shell-command to replace client

	has
	has-session -t session

	kill-server
	kill-session [-aC] [-t target-session]

	rename-session -t <N> <name>     rename session N to name

===============
-2- keybindings
===============
C-b is the DEFAULT prefix key.
The prefix key is omitted below for convenience.

_______
general
	C-b         Send the prefix key (C-b) through to the application.
	?           List all key bindings.
	d           Detach the current client.
	D           Choose a client to detach.
	:           Enter the tmux command prompt.
	f           Prompt to search for text in open windows. (doesn't seem to search, just does same as w)
	r           Force redraw of the attached client.
	t           Show the time.
	~           Show previous messages from tmux, if any.
	C-z         Suspend the tmux client.

__________
copy/paste
	[           Enter copy mode to copy text or view the history.
	]           Paste the most recent tmux-copied buffer.
	#           List all paste buffers. (All items that have been copied in copy mode.)
	-           Delete the most recently copied buffer of text.
	=           Choose which buffer to paste interactively from a list.
	Page Up     Enter copy mode and scroll one page up.

	copy/paste mode is how you would scroll up the buffer.
	Note that copy/paste mode copies to clipboard so you can paste to other
	applications etc.
	While in copy-paste mode, depending on "${EDITOR}" environment variable,
	the controls might be in emacs mode or vim mode.  Otherwise, it
	defaults to emacs.

	_____
	emacs
		C-space   set mark
		C-w       copy, (emacs is yank but doesn't actually yank)
		M-w       copy
		C-f, C-b  move forward/backward 1 char
		M-f, M-b  move forward/backward 1 word
		C-a, C-e  move to front/back
		C-n, C-p  move to next/previous line
		C-g       clear selection
		escape    cancel
		q         cancel
		g         go to line (0 is bottom, matches the n/N at topright)
		C-v       scroll down a page
		M-v       scroll up a page

	___
	vim
		space     set mark
		return    copy
		q         cancel
		escape    clear selection
		hjkl      move 1 char left down up right
		bw        move back/forward 1 word
		C-[u|d]   up/down half-page
		0$        move to front/back
		g|G       top/bottom

______________
windows (tabs)
	c     Create a new window.
	&     Kill the current window.

	0-9   Move to window N
	l     Move to the previously selected window.
	n     Move to the next window.
	p     Move to the previous window.
	w     Choose the current window interactively.
	'     Prompt for a window index to select.
	.     Prompt for an index to move the current window.
	M-n   Move to the next window with a bell or activity marker.
	M-p   Move to the previous window with a bell or activity marker.

	i     Display some information about the current window.

_____
panes
	"             Split the current pane into two, top and bottom.
	%             Split the current pane into two, left and right.

	;             Move to the previously active pane.
	o             Move to next pane.
	arrowkeys     Change to the adjacent pane in the chosen direction.

	E             equalize windows
	C-arrowkeys   Resize the current pane in steps of 1 cell.
	M-arrowkeys   Resize the current pane in steps of 5 cells.
	z             Toggle zoom state of the current pane. (moving panes toggles off zoom.)

	C-o           Rotate the panes in the current window forwards.
	M-o           Rotate the panes in the current window backwards.
	{             Swap the current pane with the previous pane.
	}             Swap the current pane with the next pane.
	M-1 to M-5    Arrange panes in one of the five preset layouts:
	              even-horizontal: all panes in a horizontal line.
	              even-vertical: all panes in a vertical line.
	              main-horizontal: pane 0: tophalf and remaining bottom half horizontal line
	              main-vertical: pane0: left half and remaining right half vertical line
	              tiled: Try to make an even NxN grid
	Space         Arrange panes in the next preset layout.

	M             Clear the marked pane.
	m             Mark the current pane (see select-pane -m).

	!             Move current pane to a new window.
	q             Briefly display pane indexes. (but no binding to select by index?)
	x             Kill the current pane.

________
sessions
	(           Switch the attached client to the previous session.
	)           Switch the attached client to the next session.
	L           Switch the attached client back to the last session.
	s           Select a new session for the attached client interactively.

________
renaming
	$     Rename the current session.
	,     Rename the current window.

-3- config

name=value          set env var
%hidden name=value  local tmux var
%if format
%elif format
%else

The config file is ".tmux.conf" in the user's home directory.

unbind keycombo         unbinds a binding.
bind keycombo command   bind a command to a keycombo
set [flags] key value   set a key/value setting
	flags:
		-s                  server setting (or use set)
		-w                  window setting (or use setw)
		-g                  global option

useful commands:
	unbind C-b              remove C-b binding so terminal can go back
	set -g prefix C-q       rebind to C-q because unused
	setw -g mode-keys vi    use vi copy mode
